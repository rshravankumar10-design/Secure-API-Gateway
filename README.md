# Sentinel API Gateway\n\nA secure, feature-rich API gateway with intelligent request filtering, rate limiting, JWT authentication, and real-time traffic analytics.\n\n## Features\n\n- **Rate Limiting**: Configure requests per minute per IP address\n- **JWT Authentication**: Secure token-based request validation\n- **IP Blocking**: Manage blocked and whitelisted IP addresses\n- **Security Levels**: STANDARD, HIGH, and PARANOID modes\n- **Real-time Analytics**: Monitor traffic, blocked requests, and latency\n- **Client Profiles**: Track user activity and risk scores\n\n## Setup\n\n```bash\nnpm install\nnpm run dev\n```\n\nThe application will start on `http://localhost:3000/`.\n\n## Build\n\n```bash\nnpm run build\n```
